# 명진빌딩 자산관리 대시보드

DSR/DSR Wire 자산관리팀 전용 대시보드입니다.  
명진빌딩(강남, 7층) 임차인 임대 현황, 전기/수도 요금, 미수금 현황을 월별로 조회하고, 세금계산서 발급내역 엑셀 파일을 자동 생성합니다.

---

## 📁 파일 구성

```
📦 프로젝트 폴더
 ├── index.html         # 대시보드 메인 (단일 파일)
 ├── bills/
 │    └── data.json     # 월별 전기/수도 요금 데이터
 ├── update.py          # 매월 데이터 자동 업데이트 스크립트
 └── 서버실행.bat        # 로컬 서버 실행 배치 파일
```

---

## 🚀 실행 방법

### 로컬 실행 (사내망)

1. `서버실행.bat` 더블클릭
2. 브라우저에서 `http://localhost:8080` 접속
3. 팀원 공유 시 `http://10.10.11.60:8080` 접속 (같은 사내망 필수)

### 외부 접속 (임시)

1. `서버실행.bat` 실행
2. 새 CMD 창에서 `ngrok http 8080` 실행
3. `Forwarding` 옆 `https://xxxx.ngrok-free.app` 주소 공유

## 📊 주요 기능

### 1. 월별 현황 조회
- 상단 드롭다운에서 조회 월 선택 (표시 기준: 청구 월)
- 예: **6월** 선택 → 6월 임대료/관리비 + **5월** 전기/수도 요금 기준

### 2. KPI 카드
- **전기요금**: 당월 전기 사용 요금 + 전월 대비 증감 + 사용기간
- **수도요금**: 당월 수도 요금 + 전월 대비 증감 + 사용기간

### 3. 층별 임대현황
- 층별 임차인명, 임대 상태, 전용면적, 임대료/관리비/전기료 표시

### 4. 전기 사용량
- 층별 월별 전기 사용량 및 요금 테이블
- 사용량 추이 꺾은선 그래프 / 층별 비교 막대 그래프

### 5. 세금계산서 발급내역 다운로드
- 사이드패널 **📑 세금계산서 발급** 버튼 클릭
- 현재 선택된 월 기준으로 자동 생성 (`.xlsx` 파일)
- **포함 항목**: 임대료, 관리비, 전기사용료 (1·3·4층만)
- **DSRK**: 분기 발급 (1·4·7·10월 양식에만 자동 포함)
- **㈜독한**: 12월 렌트프리 → 임대료 자동 제외

### 6. 미수금 현황
- 사이드패널에 **[선택 월] 마감기준 미수금현황** 표시
- **구테로이테 언주점** (1층 퇴거): 상환 스케줄 및 잔액 조회
- **독한짐** (3층 임대중): 임대료/관리비/전기료 누적 미수금 조회
- 항목 클릭 시 상세 내역 모달 표시

## 🔄 매월 데이터 업데이트 방법

매월 전기/수도 청구서를 받은 후:

```bash
python update.py
```

- 전기 검침 엑셀 파일 + 수도 청구서 PDF를 읽어 `bills/data.json` 자동 갱신
- 업데이트 후 브라우저 새로고침(F5)하면 반영
